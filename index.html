MYTHAPI

Descripción

MYTHAPI es una API RESTful diseñada para proporcionar información detallada sobre diversos personajes de mitología de todo el mundo. Esta API permite a los desarrolladores acceder a una base de datos a través de una serie de endpoints definidos.

Características

  • Personajes Míticos: Obtén información sobre héroes y figuras legendarias de diferentes mitologías, incluyendo detalles sobre sus poderes, habilidades y relaciones familiares.
  • Criaturas Míticas: Explora criaturas fantásticas y monstruos de la mitología, con descripciones de sus poderes y características.
  • Dioses y Deidades: Accede a datos sobre dioses y entidades sobrenaturales de varias religiones y mitologías, incluyendo su rol, atributos y mitos asociados.

Endpoints

  • GET /mythapi/all Lista todos los personajes.
  • GET /mythapi/id/:id Obtiene detalles de un personaje específico usando su ID.
  • GET /mythapi/nombre/:id Obtiene detalles de un personaje específico usando su nombre.

Ejemplo de Uso

Para obtener una lista de todos los personajes míticos, puedes hacer una solicitud GET a:

GET https://mythapi.onrender.com/mythapi/all

Para obtener un personaje usando su ID:

GET https://mythapi.onrender.com/mythapi/id/:id

Para obtener un personaje usando su nombre:

GET https://mythapi.onrender.com/mythapi/nombre/:nombre
